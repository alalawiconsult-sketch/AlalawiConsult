# موقع مكتب عبدالعزيز العلوي للهندسة المدنية

موقع تعريفي مجاني جاهز للرفع على GitHub Pages أو Netlify.

## طريقة التشغيل
افتح ملف `index.html` مباشرة في المتصفح.

## طريقة النشر المجاني
### GitHub Pages
1. أنشئ حساب GitHub.
2. أنشئ Repository جديد باسم مناسب.
3. ارفع ملفات الموقع الثلاثة: `index.html`, `style.css`, `script.js`.
4. ادخل Settings ثم Pages.
5. اختر Deploy from branch ثم main.
6. سيظهر لك رابط مجاني للموقع.

### Netlify
1. ادخل Netlify.
2. اسحب مجلد الموقع كامل داخل صفحة Deploy.
3. يعطيك رابط مجاني مباشرة.
